About HeartSense

HeartSense is a heart disease prediction application powered by a trained machine learning model. It accepts 13 clinical inputs and estimates the probability of heart disease, then saves those results to a dashboard for follow-up.

heart

Dataset

UCI heart disease data with target labels and common cardiology features.

Model

Random forest pipeline by default, with a saved DNN path available in the project.

Performance

Training scripts report ROC-AUC, test accuracy and 5-fold cross validation.

Features

Age, chest pain, blood pressure, cholesterol, ECG, exercise angina and more.
