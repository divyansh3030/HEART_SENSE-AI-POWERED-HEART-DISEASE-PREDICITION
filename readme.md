# HeartSense

Simple demo for heart disease prediction using scikit-learn and Streamlit.

## Setup
1. Create a virtualenv and install requirements:
   pip install -r requirements.txt

2. Place dataset CSV `data/heart.csv`. (UCI / Kaggle heart dataset)

3. Train model:
   python train_model.py

4. Run app:
   streamlit run app.py

## Notes
- This is for demonstration only — not medical advice.
- Improve by adding cross-validation, calibration, more features, and clinical evaluation.
